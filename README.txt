este é um sistema de cadastro de cliente.

no campo nome pode ser adicionado tanto o nome Completo quanto apenas o primeiro nome
no campo id é possivel criar um id alfanumerico
no campo email é apenas validado email com "@" e algo subseguente para ser validado

o cliente tem a opção de escolher entre duas alternativas; pessoa jurídica OU física
no campo de CEP entra apenas numeros 
nos campos de endereco e ruas podem ser validados Elementos alfanumerico e alfabeticos

no campo de data de nascimento aparece uma mensagem de alerta para o user entrar com o formado DD/MM/AAAA

no campo de limite de Cartao so é validado numeros
